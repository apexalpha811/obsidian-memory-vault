# Changelog

## Template — Customize Me

This is a template vault. Fork it, fill in your details, and start using it.

### Recommended first edits:
- `global/user-profile.md` — replace `{{REPLACE_ME}}` with your info
- `global/timezone.md` — set your timezone
- `agents/hermes/README.md` — add your Hermes-specific config
- `agents/claude-code/README.md` — set your repo URL in the Git Setup section
