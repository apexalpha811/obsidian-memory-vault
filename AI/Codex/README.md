# Codex Operating Hub

Default locations for Codex session notes and durable AI-assisted work.

## Folders

- `00 Inbox/` - temporary captures that need sorting.
- `01 Session Handoffs/` - end-of-session summaries.
- `02 Memory Updates/` - durable preferences, rules, and reusable lessons.
- `03 Decisions/` - cross-project decisions worth keeping.
- `04 Project Logs/` - project activity summaries.
- `05 Skill Installs/` - installed agent skills and setup notes.
- `06 Prompts/` - useful prompt patterns.
- `07 Troubleshooting/` - recurring fixes and diagnostics.
- `99 Archive/` - old AI notes.

## End / Exit Rule

When the user says `end` or `exit`, Codex should write a concise handoff, update durable memory, and add the relevant Obsidian note in this vault.
