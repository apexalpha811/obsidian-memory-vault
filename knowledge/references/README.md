# references/

Pointers to where information lives in external systems. `type: reference`.

**What to capture:** repos, dashboards, accounts, channels, document locations — what the
resource is and *when to look at it*. The pointer, not a copy of the content.

Filename: `<system>.md` (e.g. `github-repos.md`, `dashboards.md`). Tag by domain.
Never store credentials or secrets here — reference the location, not the password.
