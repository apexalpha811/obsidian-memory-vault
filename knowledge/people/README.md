# people/

One note per person — collaborators, clients, contacts, family. `type: person`.

**What to capture:** who they are, relationship to the user, how they prefer to be
contacted/worked with, recurring context. **Not** sensitive identity data (SSNs, financial
account numbers) — those never go in the vault.

Filename: `firstname-lastname.md`. Tag by domain (`[work]`, `[personal]`).
Link the people involved in a decision or project with `[[firstname-lastname]]`.
