# decisions/

Durable decisions and the reasoning behind them (lightweight ADR style). `type: decision`.

**What to capture:** the choice made, the alternatives weighed, and *why* this one won.
Future agents read these to avoid re-litigating settled questions.

Lead with the decision, then **Why:**, **Alternatives considered:**, and **How to apply:**.
Filename: `YYYY-MM-DD-short-slug.md` (date-prefixed so they sort chronologically).
Tag by domain. Link affected projects/tools with `[[name]]`.
