# Transform9 Design System

**Source:** transform9.com (live site) + Behance project by Tubik Studio
**Extracted:** May 2026

```
Use this design system for everything you create:

COLORS
Dark bg: #020801
Accent green: #A2FA8E
Medium blue: #168AE2
Dark blue: #0033CC
Light section bg: #F1F3F3
White: #FFFFFF
Muted text: #4C4C4C

Alternate dark (#020801) and light (#FFFFFF or #F1F3F3) sections. Use #A2FA8E as CTAs, accent highlights in headings, and key interactive elements.

FONTS
All: Polysans Neutral
Hero heading: 82px, weight 400
Section titles: 89px, weight 400
Large emphasis text: 98px
Body: 26px, weight 400
Nav/button: 14px, weight 400
Fallback: Arial, sans-serif

STYLE RULES
- Single font throughout — no secondary font
- CTA buttons: bg #A2FA8E, text #020801, padding 20px 24px, square (0px border-radius)
- Navigation: white at 70% opacity on dark, all lowercase
- Text links: underline only, no bg
- Highlight key words within headings with accent green (#A2FA8E)
- Clean, minimal, text-forward layout with large typography
- No gradients — solid color sections only
- Custom illustrations for hero/mascot imagery
- Warm, human, approachable healthcare-tech feel — no cold/sterile aesthetic
- Numbers/stats: 89px white on dark bg

Now apply this style to create [describe your project]. Use only the colors and fonts listed above.
```
